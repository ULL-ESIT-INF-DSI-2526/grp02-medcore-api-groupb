# MedCore REST API - Gestión Hospitalaria

El API se encuentra desplegada en Render y conectada a un clúster de MongoDB Atlas. 

**[https://grp02-medcore-api-groupb.onrender.com](https://grp02-medcore-api-groupb.onrender.com)**

Para acceder a la interfaz gráfica y probar los endpoints:
[https://grp02-medcore-api-groupb.onrender.com/api-docs](https://grp02-medcore-api-groupb.onrender.com/api-docs)

## Instrucciones de Ejecución en Local
```bash
npm run dev
npm run coverage
npm run test
```
